# 🚢 Port Management System

The **Port Management System** is a back-end Java-based web application designed to streamline and automate cargo shipping operations across global ports.

---

## 📌 Features

- 🔒 **User Authentication & Authorization** (JWT + Spring Security)
- 🧾 **Book, Track & Cancel Orders**
- 💼 **Admin Dashboard** to manage:
  - Ports
  - Ships
  - Containers
  - Orders
- 📊 **Real-Time Port Analysis Dashboard**
- 🤖 **AI-based Port Recommendation System**
  - Powered by a Python Flask microservice (scikit-learn)
- 💰 **Expense Tracking & Refund Logging**
- 📂 Modular architecture using Maven
- 🛠️ RESTful API Design with JSON communication

---

## 🛠️ Tech Stack

| Layer         | Technologies                                   |
|---------------|------------------------------------------------|
| Backend       | Java, Spring Boot, Hibernate, MySQL           |
| Security      | Spring Security, JWT, Cookie-based Auth        |
| AI Module     | Python, Flask, scikit-learn                    |
| Build Tool    | Maven                                          |
| Database      | MySQL                                          |
| Others        | Postman (API Testing), Git, GitHub            |

---

## 🚀 How to Run

### Backend (Spring Boot)
```bash
cd system
./mvnw spring-boot:run
```

### AI Recommendation Service
```bash
cd recommendation-model
python app.py
```
